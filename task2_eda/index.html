Findings

Key Findings & Data Issues

📉
Non-Normal Returns
All 12 stocks fail normality — fat tails consistent with real financial market behaviour.
🎯
Efficient Market
No stock has a statistically significant mean return — consistent with the efficient market hypothesis.
🔴
73 Price Anomalies
Z-score detection flagged 73 extreme return spikes — concentrated in the 2022 market crash period.
🏦
Sector Clustering
JPM, BAC, GS move together (high correlation). Healthcare provides natural portfolio diversification.
⚡
TSLA Most Volatile
Tesla shows highest rolling volatility but also highest average daily return (+0.164%).
📅
No Day-of-Week Effect
ANOVA confirms no significant return difference across Mon–Fri. "Monday effect" not present.

⚠️ Data Issues & Recommendations

🔍 Missing Values
60 Close + 60 Volume cells missing (0.64%)
✅ Fixed with forward-fill per ticker
⚡ Price Anomalies
73 extreme spikes detected via |Z-score| > 3
→ Winsorize at 99th percentile before modelling
📊 Non-Normal Returns
All stocks fail Shapiro-Wilk normality test
→ Use log-returns or robust statistics
📐 Beta Outliers
126 rows with Beta > 2
→ Cap at 3σ or investigate data source
💹 Negative PE
30 rows with negative PE ratio
→ Flag loss-making periods separately
