3 ML Models

All models implemented from scratch in pure NumPy — no scikit-learn or external ML libraries.

📐
Linear Regression
Ordinary Least Squares solved via the normal equation. Fast, fully interpretable, and serves as the baseline model. No iterations needed — closed-form solution.
OLS Normal Equation
🔗
Ridge Regression
L2-regularised linear model (α=1.0) that prevents overfitting across 17 features. Same normal equation solution but with a regularisation penalty added to the diagonal.
L2 Regularisation
🧠
Neural Network (MLP)
2-layer feedforward network with ReLU activations. Trained with mini-batch Adam optimiser (β₁=0.9, β₂=0.999). Xavier weight initialisation. 64 hidden units, 150 epochs.
Adam · ReLU · Xavier

Neural Network Architecture

# Input Layer
Input: 17 features → [Ret_lag1, Ret_lag2, ..., RSI, MACD, Vol_ratio, ...]

# Hidden Layer
Dense(64) → ReLU → W1: shape(17 × 64)

# Output Layer
Dense(1) → Linear → W2: shape(64 × 1)

# Optimiser
Adam | lr=5e-4 | epochs=150 | batch=64
# Loss: MSE (Mean Squared Error on next-day return)
